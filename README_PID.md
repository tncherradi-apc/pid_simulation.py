# PID Control System Simulation

This project demonstrates a PID (Proportional-Integral-Derivative) control system implemented in Python.

## Overview
A simple dynamic system is simulated and controlled using a PID controller. The goal is to reach and maintain a desired setpoint while minimizing overshoot and oscillation.

## Features
- Simulated dynamic system
- PID control implementation
- Adjustable parameters (Kp, Ki, Kd)
- Visualization of system response

## Tools Used
- Python
- NumPy
- Matplotlib

## What I Learned
- How PID controllers regulate system behavior
- How to tune parameters to improve performance
- How to simulate and visualize system dynamics
